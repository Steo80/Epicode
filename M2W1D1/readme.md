/*cos'è un algoritmo?
un algoritmo in informatica è una seguenza di istruzioni chiare e precise per  far risolvere problemi, eseguire calcoli o eseguire azioni (come raccogliere dati) e farli elaborare al computer o sul web.

cos'è una variabile?
la variabile è un contenitore (o più contenitori) al quele viene dato un nome o etichetta che serve per raccogliere i dati o i valori che devono poi venire elaborati. è chiamata variabile appunto perchè in questo contenitore  i dati o i valori possono variare di volta in volta.

undefined o null sono la stessa cosa?
no, non sono la stessa cosa;
undefined è un valore che viene dato da javascript ad un contenitore che abbiamo creato ma al quale non abbiamo ancora assegnato un valore peciso.
Null invece è un valore che abbiamo assegnato noi ad un contenitore per indicare che il contenitore non ha nessu valore. */
